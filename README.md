# 🚢 AI 船期管理系統

## 📖 系統簡介

智能化的港口船期管理系統，整合即時資料爬取、視覺化分析與泊位管理功能。

## ✨ 主要功能

- ✅ **多港口支援**: 基隆港、高雄港、臺北港
- ✅ **即時爬取**: 自動抓取最新船期資料
- ✅ **智能識別**: 自動辨識貨櫃輪
- ✅ **泊位分析**: 即時計算泊位占用情況
- ✅ **資料視覺化**: 互動式圖表與統計
- ✅ **歷史管理**: 完整的資料追蹤與查詢

## 🛠️ 技術架構

- **前端框架**: Streamlit
- **爬蟲引擎**: Selenium + BeautifulSoup4
- **資料庫**: SQLite
- **資料處理**: Pandas
- **視覺化**: Plotly

## 📊 資料來源

- 基隆港務分公司港埠資訊網
- 高雄港務分公司港埠資訊網
- 臺灣港務公司港埠資訊網

## 📝 使用說明

1. 選擇目標港口
2. 點擊「開始爬取」按鈕
3. 等待資料載入完成
4. 查看船期資料與分析結果
